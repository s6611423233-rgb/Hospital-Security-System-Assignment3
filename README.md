# ระบบบริหารจัดการความปลอดภัยโภชนาการโรงพยาบาล (Hospital Security Implementation)

**จัดทำโดย:** วศวัต กาละสัง (wasawat kalasang)  
**รหัสนักศึกษา:** 6611423233  
**วิชา:** การออกแบบและติดตั้งระบบรักษาความปลอดภัยสารสนเทศ

## รายละเอียดการติดตั้ง (Phase 2)
ระบบนี้ถูกพัฒนาขึ้นเพื่อรักษาความลับและความถูกต้องของข้อมูลคนไข้ โดยมีฟีเจอร์หลักดังนี้:

1. **AES-256 (CBC Mode):** ใช้เข้ารหัสข้อมูลออเดอร์อาหาร กุญแจถูกสร้างจาก `รหัสนักศึกษา + ชื่อจริง` เพื่อให้ค่า Ciphertext มีความเฉพาะตัว
2. **SHA-256 with Salting:** ใช้จัดการรหัสผ่านของเจ้าหน้าที่พยาบาล โดยเพิ่ม Salt จากข้อมูลนักศึกษาเพื่อป้องกันการโจมตีแบบรุ้ง (Rainbow Table)
3. **ECC (Ed25519):** ระบบ Digital Signature สำหรับยืนยันตัวตนพยาบาลผู้สั่งอาหาร เพื่อป้องกันการปฏิเสธความรับผิดชอบ (Non-repudiation)

## บันทึกการ Debug (Log Tracking)
ในการพัฒนาผมพบปัญหาดังนี้:
* **Error:** `ValueError: Data must be aligned to block boundary`
* **สาเหตุ:** ข้อมูล String ที่ป้อนเข้า AES มีขนาดไม่พอดีกับ Block size (16 bytes)
* **วิธีแก้:** นำฟังก์ชัน `pad()` จาก `Crypto.Util.Padding` มาใช้งานเพื่อเติมข้อมูลให้ครบขนาดบล็อกก่อนทำการเข้ารหัส
